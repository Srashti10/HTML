HTML Entities

Html entities are used to write those symbols which may collide with the syntax of html such as (>,<,& etc)

Common HTML Entities

        < is used for <  (3<4)
        > is used for >   (4>3)
        & is used for &   (&)
          is for non-breaking space  (new non-breaking space  )
        © is for copyright symbol     (copyright  ©)

Quotation Tag

These tags are used to give quotation. blockquote is used to give longer quotes. and q is used to give shorter quotes.
cite attribute is used inside the quotation tag to specifies the url of quote's source.

Code Tag

Code tag is a semantic tag that is used to display code snippet.

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
                <h1>This is HTML boiler plate.</h1>
            </body>
            </html>
