We can also use image in place of numbers, in the list by using the syntax: 

            list-style-image : url("linkoftheimage")

We can also use emoji in place of numbers in list by using syntax: 

            list-style-type : "emoji-here ";

We can use these properties together by using syntax: 

            list-style: disc inside;
            list-style: inside url("imagelink")
