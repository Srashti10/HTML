SEO - Search Engine Optimization.
CLS - Cumulative Layout Shift : After loading the entire page how much page is shifting. For example, if we load a website then it will take time to load fully, so may be it can happen after loading of some part , when it load fully it will shift the content of the website.
LCP - Largest Contentful Paint: It tells about how much time it will tak eby the largest element of the website. (A good LCP score is 2.5)
FID(First Input Delay): It will tell about the time in any input component needs to process such as button. A good FID is about 100 mili seconds.
LightHouse: Lighthouse generates a report on which components can be improved in HTML page.
